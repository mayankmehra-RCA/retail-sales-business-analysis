# Retail Sales Business Analysis Project

## Project Overview
This project analyzes retail sales data to identify business trends, profitability drivers, and operational insights. 
The analysis was performed using SQL and Excel to simulate real-world business analyst workflows.

## Tools Used
- SQL (MySQL)
- Advanced Excel
- Data Cleaning
- Business Analysis Techniques

## Dataset
Retail sales dataset containing 9995 transactions with 13 attributes including region, category, sales, profit, and discount.

## Key Business Questions
1. Which region generates the highest revenue?
2. Which products generate the most profit?
3. How do discounts impact profitability?
4. Which categories drive business performance?

## Key Insights
- West region generates the highest profit.
- Overall profit margin is approximately 12%.
- Some sub-categories show low profitability due to high discounts.
- Technology category drives the highest revenue.

## Business Recommendations
- Focus marketing efforts on high-performing regions.
- Reduce excessive discounting strategies.
- Promote high-margin product categories.
- Review low-performing product lines.
